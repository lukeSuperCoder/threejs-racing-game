# threejs-racing-game
一款基于threejs的赛车小游戏

## 项目介绍
### 1. 项目背景
> 该项目是基于threejs的3D游戏，游戏场景为一个简单的赛车赛道，游戏玩法为在赛道上控制汽车，汽车通过键盘控制移动，汽车通过键盘控制转向，汽车通过键盘控制加速，汽车通过键盘控制减速
### 2. 项目技术栈
> 1. threejs
> 2. 事件监听
> 3. 3D模型
> 4. 3D场景
> 5. 3D模型加载
> 6. 3D模型纹理
> 7. 3D模型材质
\*
### 3. 项目运行
> 1. 项目运行环境
    nodejs 版本 18.12.1
    npm 6.14.15
> 2. 项目运行方式
    1. 项目目录下执行命令 npm install
    2. 项目目录下执行命令 vite npx
